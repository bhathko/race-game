# Choice Race

A simple web game built with Pixi.js to help people make choices by racing characters.

## Features
- Up to 8 racers.
- Randomized speeds for a dynamic race experience.
- Track-based layout.
- Winner announcement.

## Getting Started

1. Navigate to the project directory:
   ```bash
   cd choice-race
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

## Project Structure
- `src/core/`: Core game logic and application setup.
- `src/entities/`: Game entities like `Racer`.
- `src/scenes/`: Game scenes (e.g., `RaceScene`).
- `src/config.ts`: Global configuration and constants.
